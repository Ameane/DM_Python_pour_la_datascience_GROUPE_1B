# DM_Python_pour_la_datascience_Amine_MOHAMED-BENKADDA
# Exploitation de données électorales avec Python

## Evaluation intermédiaire – Python pour la data science

Ce dépôt GitHub contient le notebook Jupyter réalisé dans le cadre de l’évaluation intermédiaire de Python pour la data science. Le sujet porte sur l’exploitation de données électorales.

## Fichiers

- `main.ipynb` : notebook principal
- `requirements.txt` : liste des dépendances Python
- `README.md` : consignes d’installation et d’exécution

## Lancer le projet

### 1. Lancement

```bash
cd DM_Python_pour_la_datascience_Amine_MOHAMED-BENKADDA
```

### 2. Installer les dépendances

```bash
python -m pip install -r requirements.txt
```

### 3. Exécuter le notebook

Ouvrir le fichier `main.ipynb` puis cliquer sur **Run All**.
